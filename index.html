더 좋은 코드를 작성하는 웹퍼블리셔 윤수민입니다.

웹 퍼블리셔는 기획 - 디자인 - 개발의 모든 포지션을 잇는 역할을 하기 때문에 웹에 대한 전반적인 이해와 폭넓은 지식을 갖추어야한다고 생각합니다.

저는 다수의 프로젝트에서 기획자로서 디자이너로서 퍼블리셔로서 참여하여 실무 경험을 쌓았습니다.

다양한 포지션에서 쌓은 실무 경험을 통해 기획과 디자인에 대한 이해도를 높일 수 있었고, 어떤 방식으로 코딩하면 데이터를 수월하게 연결할 수 있을 지에 대한 개발적인 공부도 할 수 있었습니다.

저는 마크업을 하기 전, HTML을 구조적으로 작성하기 위해 페이지의 레이아웃을 컴포넌트화 하는 작업을 먼저합니다.

탄탄한 UI 위에 CSS, javaScript 를 활용해 웹에 생명력을 불어 넣어주는 것을 좋아합니다.
